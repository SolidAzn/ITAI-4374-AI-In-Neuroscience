# Module 07 — Attention & Consciousness

This module examined selective attention — how the brain filters relevant information from noise — and its relationship to consciousness. Broadbent's Filter Model, Treisman's Attenuation Theory, and transformer self-attention were compared as competing accounts of how attention works in biological vs. artificial systems.

## Key Concepts
- Broadbent's Filter Model and Treisman's Attenuation Theory
- Inattentional blindness (Simons & Chabris, 1999)
- Cocktail party effect and bottom-up salience
- Transformer self-attention: scaled dot-product attention
- BERT attention visualization and layer-wise semantic hierarchy
- Prompt injection as an AI analog to attentional failure
