# Module 03 — Brain Anatomy & Computational Neuroscience

This module mapped brain anatomy onto AI architectures, examining how specific regions (visual cortex, hippocampus, prefrontal cortex, amygdala) inspired concrete design choices in modern deep learning. It also introduced computational neuroscience as a discipline — building mathematical models of neural systems.

## Key Concepts
- Brain region → AI architecture mappings
- Biological vs. artificial neuron structure and trade-offs
- Hodgkin-Huxley model and its descendants
- Energy efficiency, catastrophic forgetting, and continual learning
- Wolfram Language as a computational neuroscience tool
