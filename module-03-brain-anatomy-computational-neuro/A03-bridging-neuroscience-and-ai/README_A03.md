# A03 — Bridging Neuroscience and Artificial Intelligence

**Module 03 · Feb 5, 2026**

## Overview
Deep comparative analysis of biological vs. artificial neural networks — examining what was preserved, what was deliberately discarded, and what the gaps mean for the future of AI.

## Brain Region → AI Architecture Mappings

| Brain Region | Function | AI Analog |
|---|---|---|
| Visual Cortex (V1→IT) | Hierarchical feature extraction | CNNs |
| Hippocampus | Episodic memory, sequence learning | LSTMs |
| Prefrontal Cortex | Attention, working memory, goal-direction | Transformers |
| Amygdala | Emotional salience, threat detection | Risk-weighted RL modules |

## Biological vs. Artificial Neuron Comparison

| Characteristic | Biological | Artificial |
|---|---|---|
| Signal type | Electrochemical spikes | Floating-point values |
| Learning rule | Hebbian / STDP | Backpropagation |
| Energy | ~10⁻¹⁵ J/spike | ~10⁴–10⁶ J/training run |
| Adaptability | Continuous lifelong learning | Catastrophic forgetting |
| Temporal dynamics | Rich spike-timing | Static per forward pass |

## Future Directions Covered
Neuromorphic computing · Continual learning · Brain-computer interfaces · Embodied AI · Sparse models
