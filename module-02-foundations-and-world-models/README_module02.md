# Module 02 — Foundations of Neuroscience & World Models

This module explored how the brain constructs internal models of the world and how cortical regions work together through distributed, hierarchical processing. Jeff Hawkins' Thousand Brains Theory was a central framework — the idea that thousands of cortical columns each build complete models of objects and vote to reach consensus.

## Key Concepts
- Parallel distributed processing (PDP)
- Cortical columns and hierarchical temporal memory (HTM)
- World models and predictive coding
- Thousand Brains Theory — voting mechanisms for consensus
