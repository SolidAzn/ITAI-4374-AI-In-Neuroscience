# Module 05 — Sensory Processing & Perception

This module examined how the brain transforms raw physical signals into rich perceptual experience. The focus was on the visual and auditory systems — how the retina, V1, and higher visual areas process information hierarchically, and how these principles inspired convolutional neural networks.

## Key Concepts
- Center-surround receptive fields and Gabor filters
- V1 → V4 → IT cortex hierarchy (edges → shapes → objects)
- Predictive coding and top-down perceptual modulation
- Dorsal ("where") and ventral ("what") visual streams
- Cochlear frequency decomposition and auditory spectrograms
- Bio-inspired vs. learned CNN features
