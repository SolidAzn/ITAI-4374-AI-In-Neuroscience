# Module 04 — Neurons & Spiking Neural Networks

This module went deep on the spiking neuron — the fundamental unit of biological computation. The Leaky Integrate-and-Fire (LIF) model was implemented in both Python and Wolfram Language, covering membrane dynamics, spike generation, rate coding, temporal coding, and the f–I relationship.

## Key Concepts
- Leaky Integrate-and-Fire (LIF) model
- Membrane potential, threshold, reset, refractory period
- Rate coding vs. temporal coding
- f–I curves and rheobase
- Neuromorphic hardware (Intel Loihi, IBM TrueNorth)
- Spiking neural networks (SNNs) as third-generation neural nets
